IRI: http://cor.esipfed.org/ont/earthcube/srt_0000006

A semantic resource consisting of an information model represented with artifacts such as diagrams (e.g. UML, concept maps) and associated text, not formalized as an ontology.

Semantic Resource Type c

Controlled Vocabularyc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000004

A semantic resource consisting of a list of terms controlled by some authority, generally intended to populate pick lists in user interfaces.
Assumption is that users will all understand the terms without any additional guidance.
has super-classes
Semantic Resource Type c

Glossaryc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000001

A semantic resource that only consists of a set of terms and their definitions, possibly including synonyms.
Examples: NSIDC's Cryosphere Glossary https://nsidc.org/cryosphere/glossary
has super-classes
Semantic Resource Type c

Ontologyc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000003

A semantic resource containing a set of concepts with attributes and relationships between the concepts which together define a domain of knowledge, expressed in machine-readable format.
has super-classes
Semantic Resource Type c

RDF vocabularyc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000007

A semantic resource containing a representation of a knowledge domain that defines rdf classes and attributes, along with domain and range constraints for some attributes.
Distinguish from more formal ontologies by absence of general class relationships, and restrictions offered by representations like OWL.
has super-classes
Semantic Resource Type c

SKOS vocabularyc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000008

A semantic resource comprised of a collection of concepts with identifiers, preferred labels, and definitions; including broader, narrower, related and transitive relationships, labels in alternate languages, and other annotation properties.
Encoded using SKOS rdf vocabulary.
has super-classes
Semantic Resource Type c

Taxonomyc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000005

A semantic resource made up of hierarchies of terms organized as tree structures without significant relationships to concepts in other trees.
Example: domain, kingdom, phylum or division, class, genus, species is the taxonomic heirarchy in Biology.
has super-classes
Semantic Resource Type c

Thesaurusc back to ToC or Class ToC

IRI: http://cor.esipfed.org/ont/earthcube/srt_0000002

A semantic resource consisting of a structured controlled vocabulary where there is information about each term and its hierarchical, associative, or equivalence relationships to other terms in the thesaurus.
has super-classes
